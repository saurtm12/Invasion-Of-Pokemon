\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_interface_1_1_location}{Location}} is a class, which has methods dealing with the location of the objects. 

The class provides transformation from map coordinates to pixel coordinates and back, calculation of the distnce and the possibility of generating points between two coordinates.
